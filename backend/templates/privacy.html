GergVault Privacy

GergVault is an early hosted beta built from the open-source GergVault project. The goal is to help collectors catalog cards while keeping AI output, pricing, and review decisions transparent.

Account and Card Data

When you create an account or upload an intake, GergVault stores the information needed to run the app: your username, session data, uploaded card images, draft card records, metadata, review status, pricing notes, and source links.

Traffic Tracking

GergVault uses first-party operational analytics to understand traffic and debug the product. It records page/API path, method, status code, duration, logged-in user when available, session key, IP/source headers, user agent, referrer, host, and timestamp.

GergVault does not store request bodies, passwords, uploaded file contents, or form field contents inside traffic events.

Hosted Beta

The hosted version is free to start and may later add paid plans for larger collections, managed provider credits, collaboration, exports, and support. The open-source core remains available for self-hosting.
