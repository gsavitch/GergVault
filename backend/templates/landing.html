Open source

Self-hostable core

The public repo stays useful on its own: batch intake, review, card details, provider hooks, and local collection workflows. You can run it yourself and wire your own keys.

  • Django app and API
  • Draft-first card review
  • Provider-ready AI and pricing architecture
Hosted service

Managed multi-tenant vault

The hosted version uses the same foundation, then adds account isolation, managed provider setup, collaboration, backups, support, and premium collection features over time.

  • Multi-tenant hosted accounts
  • Managed AI/pricing providers
  • Future paid tiers for power collectors and teams

Free hosted beta, paid tiers later

GergVault is free while the hosted version is early. Donations help cover hosting, AI experiments, pricing provider work, and better demo data. As the hosted version matures, paid plans can support larger collections, managed provider credits, collaboration, exports, and automation.

For now, the best support is starring the repo and sharing it with collectors who have too many cards and too little time.
